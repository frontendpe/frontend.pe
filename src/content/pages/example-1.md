## ¿Qué es Frontend.pe?

Frontend.pe es una comunidad de desarrolladores frontend en Perú dedicada a compartir conocimiento, experiencias y mejores prácticas en el desarrollo web moderno.

## Nuestra Misión

Nuestra misión es:

- Fomentar el aprendizaje continuo
- Crear un espacio de networking para profesionales
- Impulsar el desarrollo tecnológico en Perú
- Compartir las últimas tendencias en desarrollo frontend

## Tecnologías que Cubrimos

En nuestros meetups y contenido cubrimos una amplia variedad de tecnologías:

- **JavaScript/TypeScript**: El corazón del desarrollo frontend moderno
- **React**: Una de las bibliotecas más populares para construir interfaces
- **Vue.js**: Framework progresivo para construir interfaces de usuario
- **Angular**: Plataforma completa para aplicaciones web
- **Astro**: Framework moderno para sitios web rápidos

## Únete a Nosotros

Si te interesa formar parte de nuestra comunidad, puedes seguirnos en nuestras redes sociales y asistir a nuestros próximos eventos.
